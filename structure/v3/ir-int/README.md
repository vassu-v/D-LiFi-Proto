# LiFi Mesh Lamp Node V3 (IR-INT)

This version integrates hardware-level IR communication using the `IRremote` library.

## Dependencies

This project requires the `IRremote` library by shirriff, z3t0. To install it using the Arduino IDE:

1.  Go to **Sketch > Include Library > Manage Libraries...**
2.  Search for "IRremote"
3.  Install the library by **shirriff, z3t0**.

Alternatively, you can install it using the Arduino CLI:

```bash
arduino-cli lib install "IRremote"
```
